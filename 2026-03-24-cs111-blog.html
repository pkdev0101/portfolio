🎮 Peppa Pig Fighting Game

▶ Play My Game

📝 Overview

This project is a JavaScript-based fighting game where the player battles different characters including George, Peppa, and Daddy Pig. The game uses object-oriented programming, APIs, and interactive controls to create a complete experience.

  • Player movement
  • Enemy battles
  • Score system
  • Leaderboard API
  • HUD updates

🧠 Object-Oriented Programming (OOP)

class Game { constructor(config) { this.config = config; this.playerName = ""; this.leaderboard = []; } startGame() { console.log("Game Started"); } endGame() { console.log("Game Over"); } } class Player { constructor(name, health, position) { this.name = name; this.health = health; this.position = position; } move(direction) { this.position += direction; } attack(enemy) { enemy.health -= 10; } }

OOP allows the game to be structured into classes such as Game and Player. Each class has properties and methods, making the code reusable and organized.

⚙️ Methods & Functions

async saveScore(score) { if (!this.apiBase) return; try { const response = await fetch(`${this.apiBase}/leaderboard`, { method: 'POST', headers: { 'Content-Type': 'application/json' }, body: JSON.stringify({ name: this.playerName, score: score, levelId: this.config.levelId }) }); if (!response.ok) { throw new Error(`POST failed: ${response.status}`); } const data = await response.json(); return data; } catch (error) { console.error("Error saving score:", error); } }

Functions allow specific tasks like saving scores. Async functions allow API communication.
